Find new: Finds a solution to a X*X queen puzzle
Find all: finds all possible solutions (single thread)
n: size of the board (n*n)
t: number of Threads to search the solution on
MT Find All: finds the number of solutions (multi-threaded search)
Benchmark: Benchmarks single thread (BS) vs multi-thread (MT) search (results in console)
BenchmarkSingle: single run of previous benchmark, use for bin board (results in console)
